Help

About
  • You are presented a 9x9 grid of tiles, further split into 3x3 grids.
  • Using the provided numbers in some of the tiles, your objective is to fill in all the other tiles.
  • You can only have 1 of each digit (from 0-9) in each row (9 spots).
  • You can only have 1 of each digit (from 0-9) in each column (9 spots).
  • You can only have 1 of each digit (from 0-9) in each 3x3 grid (9 spots).
Feedback
  • All conflicting tiles will be marked in red.
  • All tiles with given digits cannot be changed.
Number of Puzzles
  • There are currently 3 starting patterns.
  • Refreshing the page will randomly pick a pattern.
Future Updates
  • More functionality will be added in the future (such as showing possible options for each tile).
